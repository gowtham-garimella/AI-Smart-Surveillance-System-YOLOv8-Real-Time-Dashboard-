 AI Smart Surveillance System

An AI-powered surveillance system that detects objects in video streams, triggers alerts, and visualizes results in a dashboard-style interface.

⸻

🚀 Features

* 🎥 Video input via MP4 or YouTube link
* 🤖 Real-time object detection using YOLOv8
* 🚨 Smart alert system (person, car, etc.)
* ⚡ Frame skipping for performance optimization
* 📊 Detection statistics & logs
* 🧾 CSV log export
* 🖥️ Dashboard-style UI inside Google Colab

⸻

🧠 Tech Stack

* Python
* OpenCV
* YOLOv8 (Ultralytics)
* Pandas
* yt-dlp (YouTube downloader)
* FFmpeg (video compression)

* AI-Surveillance-System/
│── surveillance.ipynb
│── README.md
│── output.mp4
│── detection_logs.csv

 How to Run (Google Colab)

1. Open Google Colab
2. Copy the notebook code
3. Run all cells
4. Enter a video link (MP4 or YouTube)
